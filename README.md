# Vtoken

Signature base tokenizer

## Features

- Decrypt token with signature
- Expire in decryption

## Installation

Install vtoken with npm

```bash
  npm install vtoken
```

Install vtoken with yarn

```bash
  yarn add vtoken
```

Install vtoken with pnpm

```bash
  pnpm install vtoken
```
